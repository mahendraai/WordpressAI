=== Age Verification Popup ===
Contributors: your-name
Tags: age verification, popup, customizer, cookie
Requires at least: 5.0
Tested up to: 6.0
Stable tag: 1.0

Description:
A simple plugin that shows an age verification popup before allowing access to the site. The popup will not reappear for the same user for 5 days.

== Installation ==

1. Upload the plugin folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Customize the plugin settings via the WordPress Customizer.
